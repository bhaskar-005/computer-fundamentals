.memory hierarchy

The Computer memory hierarchy looks like a pyramid structure which is used to describe the differences among memory types. It separates the computer storage based on hierarchy.

Level 0: CPU registers
Level 1: Cache memory
Level 2: Main memory or primary memory(RAM)
Level 3: Magnetic disks or secondary memory(SSD)
Level 4: Optical disks or magnetic types or tertiary Memory

00 Registers
At the top of the hierarchy are the processor registers, which are small, high-speed memory locations built into the CPU. These are used to store data that the processor is currently working on.

01 Cache memory
supplementary memory system that temporarily stores frequently used instructions and data for quicker processing by the central processing unit (CPU) of a computer.

02 primary memory
the component of the computer that holds data , programs and instructions that are currently in use. Primary storage (RAM) is located on the motherboard.

03 secondary memory
Secondary memory is computer memory that is non-volatile, persistent and not immediately accessible by a computer or processor. It allows users to store data and information

04 Tertiary storage
Tertiary storage devices like magnetic tape are present at level 4. They are used to store removable files and are the cheapest and largest in size (1-20 TB).

.types of registers

There are several types of registers in a CPU .


01- Instruction register (IR): Stores the current instruction being executed.

02- Program counter (PC): Stores the address of the next instruction to be executed.

03- Memory address register (MAR): Holds the address of a location in memory.

04- Memory data register (MDR): Holds the data being read from or written to memory.

05- Accumulator (AC): Used to store intermediate results during the execution of instructions.

06- General-purpose registers (GPRs): Used to store data and addresses for various purposes.

07- Floating-point registers (FPRs): Used to store floating-point numbers (numbers with decimal points) for use in floating-point operations.

08- Status register (SR): Stores information about the current state of the CPU, such as whether an interrupt has occurred.

.Cache Memory

cache memory, also called cache, supplementary memory system that temporarily stores frequently used instructions and data for quicker processing by the central processing unit (CPU) of a computer.

Cache memory is used to reduce the average time required to access data from the main memory. The cache stores the most frequently accessed data, so the CPU can quickly access the data it needs without having to wait for slower main memory or storage devices to retrieve it. This can significantly improve the performance of the system.

There are typically three levels of cache memory in a computer: L1, L2, and L3.
L1 cache is the fastest cache memory and L3 cache is the slowest cache memory while L2 cache is slower than L1 cache but faster than L3 cache

Most CPUs have both L1 and L2 cache, and some also have L3 cache.
the data flows from the RAM to the L3 cache, then the L2, and finally L1. When the processor is looking for data to carry out an operation.






.Primary Memory (RAM,ROM,BIOS)

primary memory

Primary memory is a term used to describe a type of computer memory that is directly accessible by the CPU (Central Processing Unit). Primary memory is also known as "main memory" or "system memory." It is used to store data and instructions that the CPU needs to access immediately in order to execute a program.

There are two main types of primary memory: RAM (Random Access Memory) and ROM (Read-Only Memory). RAM is volatile memory, which means it is wiped clean when the computer is turned off. ROM is non-volatile memory, which means it retains its data even when the power is off.

RAM is used to store data and instructions that are being actively used or modified by the CPU. It is a fast type of memory, but it is also expensive and has limited capacity. ROM, on the other hand, is used to store instructions and data that are not expected to change, such as the BIOS (Basic Input/Output System) of a computer. ROM is slower and less expensive than RAM, but it is also less flexible because it is read-only.




RAM

RAM (Random Access Memory) is a type of computer memory that is used to store data and instructions that the CPU (Central Processing Unit) can access quickly. RAM is volatile memory, which means it is wiped clean when the computer is turned off.

RAM is an essential part of a computer, as it is used to store the data and instructions that the CPU needs to access in order to execute a program. When you open a program on your computer, the instructions and data for that program are loaded from storage (such as a hard drive) into RAM, where the CPU can access them. This allows the CPU to access the data and instructions much more quickly than if it had to retrieve them from storage, which is much slower.

RAM is a type of primary memory, which means it is directly accessible by the CPU. It is different from secondary memory, such as a hard drive or solid state drive, which is used to store data and instructions that are not being actively used by the CPU.


How data stored in RAM

RAM stands for random access memory, and it is used to store data that is being used or processed by the computer's central processing unit (CPU). When a computer is powered on, the operating system and other programs are loaded from the hard drive or SSD into RAM. The data in RAM is stored in the form of binary digits (bits), with each bit being represented by either a 0 or 1.

RAM is volatile memory, which means it is wiped clean when the computer is turned off. This is in contrast to non-volatile storage devices like hard drives and SSDs, which retain data even when the power is turned off.

RAM chips are typically made up of millions of transistors, which are tiny electronic switches that can store a single bit of data. The data in RAM is stored on the transistor as either a voltage or the absence of a voltage, with the voltage representing a 1 and the absence of a voltage representing a 0. The data in RAM is organized into memory cells, with each cell able to store a single bit.


DRAM and SRAM

SRAM faster than DRAM, but it is also more expensive to produce and requires more power to operate.

SRAM (static random access memory):

01 - Stores data as electrical charges in flip-flop circuits on a memory
02 -Data in SRAM is not refreshed like it is in DRAM, so it does not need to be constantly powered.
03 -Faster than DRAM but more expensive to produce and requires more power.
04 -Used as cache memory in CPUs and as system memory in high-end computers.



DRAM (dynamic random access memory):

01 -Stores data as electrical charges in tiny capacitors on a memory chip
02 -Capacitors in DRAM are constantly leaking their charges, so data must be refreshed periodically
03 -Slower than SRAM but less expensive to produce and requires less power
04 -Used as main memory in most computers
05 -Available in different types, including SDRAM and DDR SDRAM

.BIOS (BISO,Firmware,types of rom)

BIOS

BIOS stands for Basic Input/Output System
When a computer is powered on, the BIOS performs a power-on self-test (POST) to check that all of the hardware is functioning properly. It then looks for a bootable device, such as a hard drive or DVD-ROM drive, and loads the operating system from that device into memory.

The BIOS also provides an interface that allows the user to configure basic system settings, such as the date and time, the boot order of devices, and the system's power management settings. This is typically done through a BIOS setup utility, which is accessed by pressing a specific key during the boot process.





In modern computers, the BIOS has been replaced by the UEFI (Unified Extensible Firmware Interface), which is a more modern and feature-rich alternative to the BIOS. However, the term "BIOS" is still commonly used to refer to the firmware that is responsible for booting the computer.

kernel is the central part of an operating system that manages the system's resources and communication between hardware and software components. The kernel is responsible for controlling the system's memory, process management, and input/output operations.

Firmware is installed directly onto a piece of hardware during manufacturing.





it is typically responsible for performing a variety of tasks, such as:

02 -Monitoring the device's status and performance
03 -Providing a interface for users or other software to interact with the device

Types of ROM

There are several types of ROM (read-only memory), including:


MROM: his type of ROM stores data using magnetic states, and it has the potential to be faster and more energy efficient than other types of ROM.

PROM (programmable ROM): This type of ROM can be written to (programmed) using a special device called a PROM programmer. Once a PROM has been programmed, it is read-only and cannot be modified.

EPROM (erasable PROM): This type of ROM can be erased and reprogrammed using UV light.

EEPROM (electrically erasable PROM): This type of ROM can be erased and reprogrammed using electrical signals, rather than UV light.

Flash memory: This type of ROM can be erased and reprogrammed using electrical signals, and it is widely used in devices such as USB drives, digital cameras, and smartphones.





.Secondary Memory

Hard disks

A hard disk (also known as a hard drive or HDD) is a type of storage device that stores and retrieves digital data using rapidly rotating disks (called platters) coated with magnetic material




Structure of Hard disks

A hard drive is a physical device that stores data on a computer. It consists of a spinning disk (or disks) coated with a magnetic material, and a read/write head that reads and writes data to the disk. The disk is divided into a series of tracks and sectors, which are used to store data. The tracks are concentric circles on the surface of the disk, and the sectors are pie-shaped wedges that are used to divide each track into smaller units.

When you save a file to a hard drive, the data is stored in a specific location on the disk. The location is specified by the track and sector where the data is written. The read/write head moves to the correct location on the disk and writes the data to the surface of the disk. When you want to access a file, the read/write head moves to the correct location on the disk and reads the data from the surface of the disk.

How data stored in Hard disks

Data is stored on a hard drive by writing it to the surface of the disk (or disks) in the hard drive. The hard drive contains a read/write head that is used to write data to the disk and read data from the disk.

When you save a file to a hard drive, the data is divided into small units called sectors. Each sector can hold a specific amount of data, usually 512 bytes. The data is written to the surface of the disk in the sectors. The sectors are organized into tracks, which are concentric circles on the surface of the disk.

The read/write head moves to the correct location on the disk (specified by the track and sector where the data is to be written) and writes the data to the surface of the disk. When you want to access a file, the read/write head moves to the correct location on the disk and reads the data from the surface of the disk.

treaks

a track is a concentric circle on the surface of the disk. The hard drive's read/write head can access data on a specific track by moving to a position above the track and reading or writing data from the surface of the disk.

Tracks are used to store data on a hard drive. The hard drive's read/write head can access a specific track by moving to the correct position above the track. The tracks on a hard drive are typically organized into a series of concentric circles, with the innermost track being the first track and the outermost track being the last track.


sectors

A sector is a small unit of data storage on a hard drive's disk. It is used to store data on the hard drive and is organized into tracks on the disk. Each sector can hold a specific amount of data, usually 512 bytes. The hard drive's read/write head accesses data by moving to the correct track and sector. When you save a file to a hard drive, the data is divided into sectors and written to the surface of the disk in the appropriate track and sector. When you want to access a file, the read/write head moves to the correct track and sector and reads the data from the surface of the disk.


cluster

cluster is a group of sectors that are used to store data. Clusters are the smallest unit of storage on a hard drive and are used to store files and directories. When you save a file to a hard drive, the data is divided into clusters and written to the surface of the disk in the appropriate track and sector. The size of a cluster is determined by the size of the hard drive and the file system being used. Smaller clusters result in more efficient use of disk space, but also result in more overhead when storing small files. Larger clusters are less efficient, but have less overhead when storing small files.


cylender

cylinder is a group of tracks that are located at the same position on the disk. Cylinders are used to organize data on a hard drive and are typically used in older hard drive designs. In a modern hard drive, the read/write head can access any track on the disk, regardless of its position, so the concept of cylinders is not as important as it once was. However, the term "cylinder" is still used in some contexts to refer to a group of tracks on a hard drive.




various storage devices(magnetic tape,floppy disk)

magnatic tape

Magnetic tape is a type of storage medium that uses magnetism to store and retrieve data. It consists of a long, narrow strip of plastic film coated with a magnetic material, and a read/write head that reads and writes data to the tape. Magnetic tape is used for long-term storage of data and is commonly used for data backup and archiving. It has the advantage of being able to store large amounts of data in a small space, and it is relatively inexpensive compared to other types of storage media. However, it is slower to access data stored on magnetic tape compared to other types of storage, such as hard drives or solid-state drives.



floppy disks

A floppy disk, also known as a floppy or diskette, is a type of storage medium that was commonly used in the past to store and transfer data. It consists of a thin, flexible disk made of a magnetic material, encased in a protective plastic case. The disk is coated with a magnetic material, and a read/write head reads and writes data to the disk.

Floppy disks were widely used in the 1980s and 1990s as a means of storing and transferring data between computers.


optical disk

An optical disk is a type of storage medium that uses a laser to read and write data. It consists of a flat, circular disk made of a transparent material, such as plastic or glass, and a read/write head that reads and writes data to the disk. There are several types of optical disks, including CD (Compact Disc), DVD (Digital Versatile Disc), and Blu-ray Disc.



SD/MMC memory cards

SD (Secure Digital) and MMC (MultiMediaCard) are types of flash memory cards that are used to store data. They are commonly used in digital cameras, smartphones, and other portable devices to store and transfer data.

SD cards are a widely used standard for portable storage, and they come in several different form factors, including full-size SD cards, miniSD cards, and microSD cards. SD cards have a capacity of up to 2 TB and a maximum transfer speed of 312 MB/s.

MMC cards are similar to SD cards, but they are not as widely used. They are available in several form factors, including full-size MMC cards, miniMMC cards, and microMMC cards. MMC cards have a capacity of up to 2 GB and a maximum transfer speed of 26 MB/s.






usd and pendrive

A USB (Universal Serial Bus) is a type of interface that is used to connect devices to a computer. It is a standard way of connecting devices such as printers, keyboards, and external hard drives to a computer. A USB flash drive, also known as a pen drive or thumb drive, is a small, portable storage device that uses a USB interface to connect to a computer. It consists of a small circuit board with flash memory and a USB connector, and it is encased in a small, portable enclosure. USB flash drives are used to store and transfer data between computers, and they are widely used because they are convenient and easy to use.
